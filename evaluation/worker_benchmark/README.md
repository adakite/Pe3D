# Worker Benchmark

This benchmark compares serial (`workers=1`) and threaded auto (`workers=0`) per-grain fitting with the current default pipeline settings.

- Otira serial: 9.48 s
- Otira auto: 10.15 s
- Otira speedup: 0.93x
- Otira identical CSVs: True

- Nuage_HP serial: 121.15 s
- Nuage_HP auto: 130.88 s
- Nuage_HP speedup: 0.93x
- Nuage_HP identical CSVs: True
