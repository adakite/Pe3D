# Legacy Python Port

This directory contains an older Python port retained for parity checks and historical comparison.

It is not the maintained public package for this repository.

For public releases, use:

- `src/pe3d/` for the maintained Python package
- `archive/python_reference_v1/` for the frozen earlier baseline

Keep this directory only if you still want direct access to legacy comparison utilities such as `validation.py` and `compare.py`.
